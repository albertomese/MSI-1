### Membranator.py
Project created with the aim of placing transmembrane proteins between two slabs of dummy atoms representing a biological membrane.

Creators: Alberto Meseguer and Adrià Pérez.

This repository contains:
	-membranator.py: main program
	-structures: directory containing a set of structures in which the program has been tested
	-report.pdf: report in which we explain how the program works, some limitations and improvements of it and a brief tutorial
